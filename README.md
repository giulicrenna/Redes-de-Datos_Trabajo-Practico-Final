## Como correr el Script

*Windows:*

```bash
python -m venv .venv

.venv/Scripts/activate

pip install -r requirements.txt

python main.py
```

*Linux:*

```bash
python3 -m venv .venv

source .venv/bin/activate

pip3 install -r requirements.txt

python3 main.py
```
